Intro to Engineering

This course follows the PLTW curriculum and introduces students to the engineering process, 3d modelling, and data analysis. Students learn to document each step of the process in engineering notebooks and learn how to use different instruments like calipers, scales, and tape measures. Additionally students learn how to create 3d models using various softwares and measurements. Finally during all of these activities students learn to observe patterns and analyze data.

Marketing

Even though this class has been offered at other schools in California, this is the first year our school has offered this course. This course teaches students about different concepts such as economics, marketing, and entrepreneurship. This class also allows students to compete in any CTSO Organizations such as DECA, FBLA, HOSA, and FCCLA.

AP Computer Science A

AP Computer Science A prepares students for the AP exam held in May. The course is taught using Java as the main programming language, which allows intstructors to introduce Object Oriented Programming, algorithm development, and problem solving as a whole.

C++/Discrete

This course combines the classes discrete math and C++ programming. In this class students learn about combinatorics, algorithms, and the finite state machine. In addition they are also introduced to C++ programming and how to apply that code to real life applications.
